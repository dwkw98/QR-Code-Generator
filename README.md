# QR-Code-Generator

一个轻量级文字二维码生成器，基于 Python + Kivy 开发，支持 Android 与桌面平台。

只需输入文字，一键生成二维码，扫码即可读取内容。

## ✨ 功能特点

- ✅ 输入任意文字（支持中文），即时生成二维码
- ✅ 极简操作，无广告，纯本地生成，不联网
- ✅ 自动适配 Android 系统字体，中文显示清晰
- ✅ 生成的二维码可截图保存或分享
